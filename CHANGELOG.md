# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - {PR_MERGE_DATE}

### Added
- Unified search across Chrome tabs, bookmarks, and browsing history
- Automatic Chrome profile detection (reads from Local State)
- Smart tab switching — switch to existing tabs instead of opening duplicates
- Favicon display for quick visual identification
- Colored tags to distinguish result sources (Tab, Bookmark, History)
- URL deduplication with priority: tabs > bookmarks > history
- Manual refresh via ⌘ + R
- Copy URL action via ⌘ + C
- Open in default browser action
